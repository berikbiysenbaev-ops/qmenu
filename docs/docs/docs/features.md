# QMenu Features

## Client
- QR code scan
- Language selection (Kazakh/Russian)
- View menu
- Search menu
- Add to cart
- Edit cart
- Place order
- View order status
- Call waiter
- Request bill

## Waiter
- Receive new orders
- Accept orders
- View active tables
- Receive waiter calls
- Receive bill requests

## Kitchen
- View incoming orders
- Mark as Preparing
- Mark as Ready

## Admin
- Manage menu
- Manage categories
- Manage tables
- View statistics
- Manage staff

# QMenu

QMenu — QR код арқылы мейрамхана ішінде тапсырыс беру жүйесі.

## Жоба мақсаты

Клиент QR кодты сканерлейді.

Мәзір ашылады.

Тағам таңдайды.

Тапсырыс береді.

Тапсырыс официант пен асүйге бірден түседі.

## Модульдер

- Клиент
- Официант
- Асүй
- Әкімші
